# Sinalizadores de integração no aplicativo {#integrate}

Esta seção fornece diretrizes de integração para todos os clientes de Sinalizadores, organizados por tipo de aplicativo.

## Escolha seu caminho de integração {#choose}

Selecione o guia que corresponde ao tipo de aplicativo:

1. [Guia de inicialização](startup-guide.md) — Comece aqui para obter uma visão geral de alto nível de todas as etapas de integração
1. [Aplicativos móveis](mobile-applications.md) — Integração do SDK para aplicativos móveis
1. [Aplicativos Web](web-applications.md) — Integração do SDK para propriedades da Web

## Principais tipos de aplicativos {#overview}

| Tipo de aplicativo | Integração recomendada |
| --- | --- |
| **Aplicativo móvel** | AEP Mobile SDK — consulte os guias do [Android](../sdk-releases/android/android-extension-integration-guide.md) e [iOS](../sdk-releases/ios/ios-extension-integration-guide.md) |
| **Aplicativo Web** | Web SDK — consulte [guias de integração da Web](../sdk-releases/web/web-extension-integration-guide.md) |
