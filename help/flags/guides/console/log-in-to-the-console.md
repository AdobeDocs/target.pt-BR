# Sinalizadores de acesso no Adobe Target {#log-in}

Para começar a usar Sinalizadores, solicite acesso e abra os Sinalizadores no Adobe Target.

## Solicitar acesso {#request-access}

Solicite acesso por meio do sistema de gerenciamento de acesso de sua organização. Consulte [Solicitar acesso](request-access.md) para obter instruções passo a passo.

Depois que sua solicitação for aprovada, você receberá as permissões associadas à função concedida pelo administrador.

## Sinalizadores de Acesso {#log-in-steps}

Após o acesso ser concedido:

1. Vá para [https://experience.adobe.com/](https://experience.adobe.com/) e entre com as credenciais da organização.
1. Abra o **Adobe Target**.
1. Selecione **Sinalizadores** na navegação do Destino.
1. Selecione a sandbox correta. Consulte [Visão geral dos ambientes](environments-overview.md) para obter detalhes.

## Primeiras etapas após acessar sinalizadores {#first-steps}

Depois de abrir os Sinalizadores, verifique se o aplicativo está listado. Os aplicativos são gerenciados por administradores. Se seu aplicativo não estiver listado, entre em contato com seu administrador para adicioná-lo. Consulte [Gerenciar aplicativos](../applications/manage-applications.md) para obter mais informações.

## Terminologia principal {#terminology}

| Termo | Definição |
| --- | --- |
| **Aplicativo** | O aplicativo que você deseja controlar com sinalizadores de recursos. |
| **Sinalizador de recurso/Grupo de recursos** | Os artefatos criados em Sinalizadores para testes de recursos e implantações controladas. |
