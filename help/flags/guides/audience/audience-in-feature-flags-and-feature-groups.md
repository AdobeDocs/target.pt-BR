# Público-alvo em sinalizadores e grupos de recursos {#audience-overview}

Os sinalizadores fornecem direcionamento de público-alvo para sinalizadores de recursos e grupos de recursos. Você pode combinar variáveis de contexto para controlar com precisão quais usuários recebem um recurso.

Para direcionamento baseado em contexto (por exemplo, direcionamento pelo idioma ativo ou tipo de dispositivo do usuário), consulte [Usar contexto nas regras de público-alvo](using-context-in-audience-rules.md).
