# Atividade do Recomendações

As atividades do Recommendations no Adobe Target exibem automaticamente produtos ou conteúdo que podem ser do interesse dos clientes com base em atividades do usuário anteriores. O Recomendações ajuda a direcionar os clientes para itens relevantes que podem ser novidade para eles.
