# [!UICONTROL Alterações do Visual Experience Composer]

A versão [!DNL Adobe Target Standard/Premium] 25.2.1 (17 de fevereiro de 2015) apresenta o [!UICONTROL Visual Experience Composer] (VEC) atualizado. Este artigo explica as diferenças entre as versões anteriores e atualizadas do VEC.

>[!IMPORTANT]
>
>O [!UICONTROL Visual Editing Composer] atualizado requer a [!DNL Adobe Experience Cloud] [[!UICONTROL extensão de Auxiliar de Edição Visual]](/help/main/c-experiences/c-visual-experience-composer/r-troubleshoot-composer/visual-editing-helper-extension.md) disponível no [!DNL Chrome Web Store].

O VEC é exibido ao criar ou editar uma atividade existente.

![Visual Experience Composer (VEC)](/help/main/c-experiences/c-visual-experience-composer/assets/vec-highlight-refresh.png)

## Principais alterações no VEC

As seções a seguir explicam as principais alterações no VEC atualizado em comparação à versão anterior.

### Painel [!UICONTROL Experiências]

Como na versão anterior, o painel [!UICONTROL Experiências] permanece no lado esquerdo do VEC. O painel [!UICONTROL Experiências] não pode ser recolhido.

![Painel de experiências](/help/main/c-experiences/c-visual-experience-composer/assets/experiences-panel.png)

Você pode criar, renomear ou remover experiências usando o painel [!UICONTROL Experiências]. Clique no ícone **[!UICONTROL Adicionar]** ( ![Ícone Adicionar](/help/main/assets/icons/Add.svg) ) para adicionar uma nova experiência. Clique no ícone [!UICONTROL Mais Ações] ( ![Ícone Mais Ações](/help/main/assets/icons/MoreSmall.svg) ) para duplicar, excluir ou redirecionar uma experiência.

### Painel [!UICONTROL Componentes] (novo)

Você pode adicionar vários componentes à sua página da Web e editá-los conforme necessário usando o painel novos [!UICONTROL Componentes].

![Painel de componentes](/help/main/c-experiences/c-visual-experience-composer/assets/components-panel.png)

Para adicionar um novo componente, selecione o componente desejado no painel [!UICONTROL Componentes], passe o mouse sobre um elemento existente na página e escolha inserir o componente antes de ou depois do elemento selecionado.

>[!NOTE]
>
>Se você vir o painel [!UICONTROL Modificações] nessa área em vez do painel [!UICONTROL Componentes], clique no ícone **[!UICONTROL Mostrar Componentes]** ( ![ícone Mostrar Componentes](/help/main/assets/icons/Add.svg) ). O ícone [!UICONTROL Mostrar Componentes] ( ![Ícone Mostrar Componentes](/help/main/assets/icons/Add.svg) ) e o ícone [!UICONTROL Mostrar Modificações] ( ![Mostrar painel Modificações](/help/main/assets/icons/History.svg) ) atuam como alternadores para mostrar as opções apropriadas.
>
>Para recolher o painel [!UICONTROL Componentes] e ampliar a tela [!UICONTROL Design], enquanto o painel [!UICONTROL Componentes] estiver aberto, clique no ícone ( ![Mostrar Componentes](/help/main/assets/icons/Add.svg) ).

### Painel [!UICONTROL Modificações]

Para abrir o painel [!UICONTROL Modificações], clique no ícone [!UICONTROL Mostrar modificações] ( ![Mostrar painel Modificações](/help/main/assets/icons/History.svg) ) no painel [!UICONTROL Componentes]. O painel [!UICONTROL Modificações] mudou de posição do lado direito para o lado esquerdo da tela de edição.

![Painel de modificações](/help/main/c-experiences/c-visual-experience-composer/assets/modifications-panel.png)

>[!NOTE]
>
>O ícone [!UICONTROL Mostrar Componentes] ( ![Ícone Mostrar Componentes](/help/main/assets/icons/Add.svg) ) e o ícone [!UICONTROL Mostrar Modificações] ( ![Mostrar painel Modificações](/help/main/assets/icons/History.svg) ) atuam como alternadores para mostrar as opções apropriadas.
>
>Para recolher o painel [!UICONTROL Modificações] e ampliar a tela [!UICONTROL Design] enquanto o painel [!UICONTROL Modificações] estiver aberto, clique no ícone [!UICONTROL Mostrar Modificações] ( ![Mostrar painel Modificações](/help/main/assets/icons/History.svg) ).

O painel [!UICONTROL Modificações] mostra todas as alterações feitas em sua página no VEC e permite que você faça alterações adicionais (como Seletor de CSS, Mbox e Código personalizado).

Clique no ícone [!UICONTROL Mais Opções] ( ![Ícone Mais Ações](/help/main/assets/icons/MoreSmall.svg) ) para adicionar uma modificação, excluir todas as modificações ou excluir todas as modificações inválidas. Clique em [!UICONTROL Selecionar] para executar operações em massa: [!UICONTROL Aplicar a Todas as Páginas] ou [!UICONTROL Excluir].

Para exibir o painel [!UICONTROL Modificações] novamente, clique no ícone [!UICONTROL Ocultar Modificações] ( ![Mostrar painel Modificações](/help/main/assets/icons/History.svg) ) no painel [!UICONTROL Modificações].

### Painel [!UICONTROL Propriedades] (novo)

O painel [!UICONTROL Propriedades] permite alterar as propriedades dos elementos selecionados na página, sejam eles elementos do HTML ou objetos específicos do [!DNL Target], como recomendações ou ofertas.

![Painel de propriedades](/help/main/c-experiences/c-visual-experience-composer/assets/properties-panel.png)

Clique nos ícones na parte superior do painel para editar o código HTML ou excluir, duplicar ou ocultar elementos. As alterações aparecem no painel [!UICONTROL Modificações].

![Ícones de propriedade](/help/main/c-experiences/c-visual-experience-composer/assets/options-icons.png)

O painel [!UICONTROL Propriedades] pode ser recolhido no painel direito. Clique no ícone [!UICONTROL Mostrar/Ocultar Propriedades] ( ![ícone Propriedades](/help/main/assets/icons/Propertie.svg) ) à direita do painel para recolher ou exibir o painel [!UICONTROL Propriedades].

### Configurações/configurações de atividade

Clique no ícone [!UICONTROL Configurar] ( ![Ícone Configurar](/help/main/assets/icons/Setting.svg) ) exibido na parte superior da tela de design para exibir o menu de propriedades da atividade.

![Opções de configurações da atividade](/help/main/c-experiences/c-visual-experience-composer/assets/configure-options.png)

As diferentes opções permitem atribuir propriedades, editar regras de entrega de página, especificar preferências do site, adicionar outras páginas e ativar ou desativar atividades de várias páginas ou de vários públicos-alvo. Atribuir [!UICONTROL Propriedades] é um recurso [[!DNL Target Premium]](/help/main/c-intro/intro.md#premium).

A posição e a funcionalidade são semelhantes à interface anterior do VEC.

### [!UICONTROL Design]/[!UICONTROL Procurar] modos

Use os botões [!UICONTROL Design]/[!UICONTROL Procurar] exibidos na parte superior do painel [!UICONTROL Propriedades] para alternar entre o modo de design e de navegação.

![Alternância de design e navegação](/help/main/c-experiences/c-visual-experience-composer/assets/design-browse-mode.png)

Use o modo [!UICONTROL Procurar] para navegar no site e escolher o modo de exibição ou a página que deseja atualizar. Retorne ao modo [!UICONTROL Design] para adicionar ou editar suas alterações.

### [!UICONTROL Desfazer]/[!UICONTROL Refazer]

Você pode desfazer as alterações feitas ao clicar no ícone [!UICONTROL Desfazer] ( ![ícone Desfazer](/help/main/assets/icons/Undo.svg) ).

![Ícone Desfazer no VEC](/help/main/c-experiences/c-visual-experience-composer/assets/undo.png)

Para refazer uma ação, expanda o grupo de botões Desfazer/[!UICONTROL Refazer] e escolha [!UICONTROL Refazer].

### Tela [!UICONTROL Design]

A tela [!UICONTROL Design] permite selecionar janelas de visualização, inclusive ajustar à tela, [!UICONTROL Área de Trabalho], [!UICONTROL Tablet], [!UICONTROL Paisagem Móvel] e [!UICONTROL Retrato Móvel].

![Opções de viewport](/help/main/c-experiences/c-visual-experience-composer/assets/viewports.png)

O VEC atualizado também permite aumentar ou diminuir o zoom clicando no ícone apropriado ( ![ícone de Aumentar o Zoom](/help/main/assets/icons/ZoomIn.svg) ou ![ícone de Diminuir o Zoom](/help/main/assets/icons/ZoomOut.svg) ).
